# Persists Realms 持续领域

<h3>
  一个实验性的、开源的、公益化的 Minecraft 服务器社区
</h3>


## 这是什么？

Persists Realms 持续领域（前 BotWMCS）是一个公益性的 Minecraft 服务器社区团体，我们做 Mojang 没有做的事情。

目前，我们致力于完善一个原版化的养老生存主题服务器，并且我们正在开发针对多人游戏的模组，比如 [BotWMCS RealmsHost](https://github.com/BotW-Minecraft-Server/RealmsHost)。

:::caution 注意事项
  我们并不对 Minecraft 进行任何逆向工程研究，我们只是一个 Minecraft 模组开发团队和服务器团队。
:::

### 关于 BotWMCS 和 Persists Realms

BotWMCS 是一个由 2021/05/01 开始的 Minecraft 养老生存服务器。起初这只是一个好友之间的联机游戏，而后由于一则 [Bilibili 视频](https://www.bilibili.com/video/BV11o4y127DP)意外地火爆，从而经历了 6 个大版本，演变成了一个庞大的模组服务器。

在服务器人数逐渐增多的情况下，起初为联机游戏设计的游戏模式逐渐无法应对庞大的玩家群体。在经过第 6 个大版本之后，Sam_Chai（BotWMCS 服主）决定，重新制作 BotWMCS。于是，本就针对多人游戏的 Persists Realms 诞生了。

#### Persists Realms

Persists Realms（简称 PR，又或者为 持续领域），是脱胎于 BotWMCS 但又不继承 BotWMCS 的全新的服务器。内部版本号为 Patch 7，但由于迫切地想让玩家了解持续领域为一个全新的、与 BotWMCS 卡顿且臃肿的服务器分割开来的，持续领域引入了一个全新的版本号，即 LTSX。

### 重新制作与考量

*目前，传统的 Minecraft 服务器仍然划分为两个大的群体，即插件服和模组服。BotWMCS 作为一个典型的模组服务器，并没有体现大型服务器应该有的管理制度和模式，在服务器人数近乎破百的情况下，竟然仍没有开发出较为完善的权限系统，这使得玩家们的权限不受严格控制，引发了许多的争端。*

玩家们的争端使得 BotWMCS 被加速地推进了墓碑中，在 BotWMCS 寿终正寝之际，Sam Chai 计划推出一个全自主研发模组的服务器。这也是 LTSX 的雏形。

全自主研发的服务器被快速地提上日程，但 Sam Chai 与全体的服务器玩家高估了这一开发的进度。由于 BotWMCS 后期橡皮图章似的管理组和名不副其实的开发组，使得 LTSX 全自主研发计划一拖再拖，导致玩家流失惨重。于是，Sam Chai 决定重新考量这一计划的必要性，在听从其余老玩家们的建议后，决定以半研发模式推进这一计划。这也是 Persists Realms 目前做的工作。

### 目前的开发进度...

正如预想中的那样，Persists Realms 目前正在尝试走出历史的泥潭中，以一个全新的服务器彻底抛弃 BotWMCS 的历史包袱。

* **全新的版本与模组列表。**持续领域目前正在 1.20.1 Fabric 平台开发，采用与 BotWMCS 模组不同、但游玩方法大致相同的模组。这包括 ReTerraForged、Let's Do 系列的模组。
* **通过自主研发模组为 Fabric 服务器提供多人游戏可用性。**以往模组服务器都需要 Hybrid 服务端提供 Bukkit API 来满足服务器的管理系统所需，而通常这种服务端的兼容性和稳定性都很差。持续领域的目的是提供一个高可用、流畅的游玩环境，所以 Sam Chai 决定将自研模组开发重心移至服务器侧以保证纯 Fabric API 也有类似的管理系统。
* **一个设想：在移动平台运行持续领域客户端。**在遥远的将来，持续领域或许可以走进玩家的手机中，通过*~~某些魔法技术~~*来打通基岩版模组服的隔阂。
* **为其他模组服务器提供一个点子。**持续领域在纯模组服的道路上一路深耕，造就了 [RealmsHost](https://github.com/BotW-Minecraft-Server/RealmsHost)。该模组深度改写了 Fabric 服务端核心，提供了类似 Bukkit API 的权限管理系统。

## 目前开服了吗？

很抱歉，还没有。

目前由于 Sam Chai 个人的情况，已经不能有全部的时间去打理与开发持续领域，但目前即使进度缓慢，也仍在进行当中。

## 关于 Persists Realms 的开发组

### 我们都在干什么？

这已经在[开头](#这是什么)解释了。

### 我们在开发哪些内容

基于 Fabric 的用于构建大型多人游戏服务器的模组。如为玩家提供数字货币的模组、权限节点模组、领地模组、小镇、公会模组、经济系统和商店模组等等。

此外，我们也会考虑开发独一无二的玩法性模组。

### 目前都有什么项目

可以从 [GitHub](https://github.com/BotW-Minecraft-Server) 查看。

### 我们的现状

正如上文所述，目前即使是开发组，也陷入发展缓慢阶段，这由于目前的开发组仍然以 Sam Chai 为主导的开发工作。目前，Sam Chai 正在考虑将开发组领导者替换为 jianzoushihu（BotWMCS 的老玩家，目前正在为持续领域贡献自己的成果）
